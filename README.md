# Практика

Репозиторий бригады 9 (группа 25ВВВ3) по учебной практике 1 курса. Вариант: **поразрядная сортировка**.

## Описание задания

В рамках практики необходимо:

Разработать программу на языке C, реализующую алгоритм сортировки из варианта, с описанием разработки и тестирования.

Программа реализует ввод данных через `scanf`, арифметические операции, ветвление (`if / else`) и работу с файлами (создание/открытие, чтение, запись, закрытие).

Программа содержит информационное окно ввода и вывода.

В программе реализовано полноценное тестирование с подсчётом времени выполнения для тестирования на разных входных данных.

Программа реализует полноценную БД случайных чисел.
